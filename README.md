# LifeFlow AI 💉
An AI-based blood donation management system that predicts future blood needs.

## 🚀 Tech Stack
- Next.js
- Node.js
- PostgreSQL
- Prisma ORM
- Redis for caching

## ✨ Features
- Real-time donor tracking
- Predictive model using machine learning
- Secure authentication (JWT)
